Этот проект представляет собой **WPF-приложение** для управления магазином электроники, разработанное с использованием **Entity Framework Core** и паттерна **Code First/Database First**. Интерфейс выполнен в стиле **Material Design**.

## Основные возможности

* **Авторизация**: Система входа с разделением ролей (Посетитель и Менеджер через PIN-код "1234").
* **Управление товарами**:
* Просмотр списка товаров с динамической фильтрацией по названию, цене, категории и бренду.
* Сортировка по цене, названию и количеству на складе.
* Добавление, редактирование и удаление товаров (доступно менеджеру).


* **Справочники**: Управление категориями, брендами и тегами в отдельном интерфейсе управления.
* **Валидация**: Поля ввода (цена, рейтинг, склад, названия) проверяются на корректность с выводом ошибок пользователю.
* **Визуальные эффекты**: Цветовая индикация товаров с низким остатком на складе (менее 10 единиц) через конвертеры данных.

## Технологический стек

* **Платформа**: .NET 8.0 Windows (WPF).
* **База данных**: Microsoft SQL Server.
* **ORM**: Entity Framework Core 9.0.11.
* **UI Framework**: Material Design In XAML Toolkit.

## Структура базы данных

Проект использует реляционную модель данных, включающую таблицы:

* `Products` (Товары).
* `Categories` (Категории).
* `Brands` (Бренды).
* `Tags` (Теги) и связующая таблица `Product_Tags` для реализации связи многие-ко-многим.
* `Users` (Пользователи).
